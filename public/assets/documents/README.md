# Documenti pubblici EuropaService

Questa cartella contiene solo documenti pubblici scaricabili o mostrabili sul sito.

Regole:
- Non inserire fatture, bonifici, FIR, ricevute, documenti contabili o documenti interni.
- Non inserire dati personali, dati cliente, indirizzi privati o dettagli economici riservati.
- I documenti interni andranno in futuro in storage privato, non in `public`.
- Usare nomi file descrittivi e stabili.
- Se un documento non e' ancora disponibile, mantenerlo nei dati come "in preparazione".
