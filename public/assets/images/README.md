# Immagini pubbliche EuropaService

Usare solo immagini stock con licenza adatta all'uso commerciale, ad esempio da Unsplash, Pexels o Pixabay.

Regole:
- Non usare immagini prese a caso da Google.
- Evitare loghi, marchi, targhe, insegne riconoscibili e persone riconoscibili.
- Salvare i file in cartelle coerenti: `hero`, `services`, `projects`, `sectors`, `team`, `placeholders`.
- Usare nomi file descrittivi, ad esempio `controsoffitto-cartongesso-hotel.jpg`.
- Compilare sempre alt text descrittivi nei dati del sito.
- Verificare licenza, fonte e credito prima della pubblicazione reale.
